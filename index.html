Guide

How the hiring pipeline works and scoring reference

Pipeline Stages
1
Application Received
Resume comes in. HR creates applicant record and reviews for basic fit.
2
Assessments Sent
One email with DISC link + GRIT form + Values form + blank EMM file.
3
Results Recorded
HR enters DISC result and pastes GRIT/Values scores. Auto-pulled from Google Sheets if connected.
4
EMM Grading
Upload applicant xlsx, auto-grade, review integrity flags. Pass threshold is 75%.
5
Interview
Live validation scorecard with 5 questions. Final hiring decision recorded.

Google Sheets Sync
Not connected

Once connected, all applicant records sync automatically to your Google Sheet so every HR staff member sees the same data in real time. GRIT and Values scores are pulled in automatically by email match.

Setup Guide
1
Open your master Google Sheet
Click here to open the ABBSS Hiring Pipeline Sheet
2
Open Apps Script
In the Sheet click Extensions then Apps Script. A new tab opens with a code editor.
3
Paste the script
Select all existing code (Ctrl+A), delete it, click Copy Apps Script Code below, paste it, then save (Ctrl+S).
4
Deploy as Web App
Click Deploy then New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.
5
Paste URL above and connect
Paste the URL in the field above and click Save and Connect. Done.
